    📖 新手指引 · 把 PaperPiggy 养成你的专属知识库
    1

    PaperPiggy 是什么

    它是你的本地文献知识库：把论文 PDF 喂进来，它替你精读、秒级检索、按页码引用、写综述。文献始终留在你电脑上，不上传任何服务器。

    三种用法，由轻到重：① 在应用里直接检索；② 用内置对话问（填一个大模型 Key）；③ 接进 🤖 AI 助手（Claude Code 等），让它替你系统性地查库、写作、养库。

    2

    第一步 · 把文献喂进来

    两种来源，二选一（首次会有向导带你走）：
    · Zotero：直接读你的 Zotero 库，新增文献后自动增量入库。
    · 文件夹：把 PDF 放进一个文件夹（或直接拖进窗口），AI 自动抽题录入库。
    入库后题录秒建，马上能按标题 / 作者 / 摘要搜到。

    3

    第二步 · 让文献可深读（① 深索 → ② 检索摘要）

    ① 深索＝把 PDF 全文拆成可检索的小段。只有深索过的文献才能被精读、页级引用、跨篇综合；未深索的只有题录 + 摘要。在「📚 浏览」里挑文献深索，建议慢慢把常用的喂完——可放后台跑，不影响使用。

    ② 检索摘要（深索之后的一步）——再给每篇深索文献配一段 ~150 字 AI 摘要（⚙ 设置 → 检索摘要），让检索更容易找准这篇。可让 Agent 跑深索时顺带生成（省 API 额度），或在「📚 浏览」对单篇点「⚪ 无摘要」现生成、或在库总览点「🧬 生成检索摘要」批量补。

    4

    第三步 · 检索与提问

    · 「🔍 检索」：抛一个研究问题，全库秒找相关文献与原文段落，按期刊权威度排序、精确到印刷页码。
    · 「💬 对话」：不想自己翻，就用大白话问，它读你的库回答、每句可回溯到来源页（需在对话页填一个大模型 API Key，硅基流动等有免费额度）。

    5

    第四步 · 养库（越用越厚）

    好答案不该说完就忘。它能把结论沉淀成「📖 综述库」——带页码引用、可复用、下次直接命中；还能记住你的偏好与进度（记忆），甚至定期帮你搜集动态（定时任务）。喂得越勤，小猪越懂你。

    6

    内置「对话」 vs 「🤖 Agent」，怎么选？

    💬 内置对话随手问答、带页码；一次看少量来源；答案默认不自动沉淀。轻量、开箱即用。
    🤖 接 AI 助手跨会话累积、系统化综述、写回交付物 / 综述 / 记忆、定时任务。更强，适合做研究、写论文。

    两条路都可用，按需选。想深入用 Agent，去「🤖 Agent」页点开那里的完整教程。

    7

    自动更新、备份与隐私

    · 自动更新：在「⚙ 设置 → 自动更新知识库」可设按天定时增量入库（默认每天 7:00）；也能随时点顶栏「⟳ 手动更新知识库」。
    · 备份：「⚙ 设置 → 备份与恢复」把丢了就再也没有的东西（综述、Agent 写的论文与记忆、收藏夹、期刊分级）打成一个 zip。想要云备份：把备份位置指到 OneDrive / 坚果云的文件夹里——同步一个静态 zip 是安全的。⛔ 但别把整个数据目录丢进云盘：向量索引是持续读写的数据库，云盘同步它早晚会把索引搞坏。
    · 隐私：全程本地，文献不出你的电脑；大模型 Key 只存本机浏览器与本地服务内存，不上传第三方（除你自己选的模型服务商）。备份包默认不含 API 密钥（要含得自己勾选）。

    8

    小贴士

    · 顶栏这几个页签可以拖动调整顺序，把最常用的放前面。
    · 想重看这份指引：随时点首页「📖 新手指引」；想重走安装向导：「⚙ 设置」里有入口。
    · 深索是慢功夫——先深索你最关心方向的文献，问答立刻更扎实。

    ✍️ 写论文与综述端到端帮你写：查库 → 提纲 → 起草 → 按页码核验引注 → 沉淀。说「基于我的库写一篇关于…的综述」。

      5

      综述库（wiki）：把问答沉淀成越用越厚的知识

      AI 每答一个问题、读完一篇文献，可把结论写成一页带引用的综述，下次直接命中、不重复劳动。它会主动维护：新文献可能推翻旧结论时会标记「过时」并写明理由。你放心——AI 写的页都标 🤖 未核验、不覆盖你亲自核验过的页、随时可一键删。而且这些未核验页在检索里会被降权，不会盖过你的原始文献；你亲自核验过的页则恢复正常权重——AI 一眼就能分清哪些是它自己没核过的草稿、哪些是可信综合。在主菜单「📖 综述库」页可浏览、核验、删除。

      6

      深索：让回答精确到页码

      已深索的文献 AI 能读到精确页码的正文；未深索的只有题录 + 摘要。想某方向问答更扎实，先去「📚 浏览」深索它（拆成可检索的小段，可后台慢慢跑，不影响使用）。

      💡 第②步：检索摘要——深索完，让 AI 给每篇写一段检索摘要（⚙ 设置 → 检索摘要），检索更容易命中这篇。可让 Agent 跑深索时顺带生成、或在「📚 浏览」对单篇点「⚪ 无摘要」、或在库总览「🧬 生成检索摘要」批量补。
