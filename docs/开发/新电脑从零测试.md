# 新电脑从零试用 —— 操作 & 排查清单

> 目的：在一台干净的 Windows 电脑上，从零走一遍完整体验，揪出"只在开发机成立"的问题。

## 你拿到的测试包
一个自包含文件夹 `LocalKB\`（约 2GB）：内嵌 Python + 应用 + **已带 slim 模型**（bge-m3 + reranker 量化版）+ `portable.txt`（数据/模型都放在包内，不写系统盘）。
> 为什么带模型：正式版是"首启云端下载"，但模型上传 GitHub 还没做，所以测试包直接内置，保证**从零离线可跑**。

## 新电脑前置条件
- Windows 10/11 **64 位**。
- 装了 **Zotero** 且里面有文献（这样才能测出真实建库；没有也能用 .bib 测）。
- WebView2 运行时（Win10/11 一般自带；没有会自动回退系统浏览器）。

## 步骤
1. 把整个 `LocalKB\` 文件夹拷到新电脑（U 盘/移动硬盘/OneDrive 均可），**放桌面或某个可写目录**（别放 Program Files）。
2. 双击 **`启动.bat`**（或 `LocalKB.vbs` 无黑窗）。首次会弹出原生窗口 + 首启向导。
3. 向导五步：
   - **① 环境自检**：应显示探测到 Zotero 目录、语义/重排模型"已就绪"。
   - **② 选检索引擎**：测试包选 **本地模式**（模型已带，直接下一步）；想测 API 就填个 SiliconFlow 免费 key。
   - **③ 连接文库**：应自动认到你的 zotero.sqlite，显示条目数。
   - **④ 准备引擎**：本地模型已在 → "引擎已就绪"，直接下一步。
   - **⑤ 即时索引**：几秒建好 L 词法索引 → 完成，进主界面。
4. 试功能：🔍检索（关键词/语义）、📚浏览（收藏夹+AI主题）、📊库总览、💬对话（需在设置里填 LLM key）。
5. 深索一两篇：浏览里勾选文献→深索，或库总览的深索卡。看能否提取 PDF、建向量、之后检索出页级正文。

## 重点看这些（最可能出问题）
- [ ] 双击 `启动.bat` 能否弹出**原生窗口**（还是黑窗/报错/回退浏览器）。
- [ ] **Zotero 自动探测**成不成功（你的数据目录若在非默认盘，考验 prefs.js 探测）。
- [ ] **从零建 L 索引**成不成、几秒完成。
- [ ] **检索**能否返回结果、页码/期刊分级对不对。
- [ ] **深索一篇 PDF**：提取→嵌入→检索全链路（考验包内 onnxruntime + pymupdf）。
- [ ] 中文路径/中文界面有无乱码。
- [ ] 关窗口后进程是否干净退出（任务管理器看有无残留 python）。

## 已知未做（这次不用测）
