# 知识库 · 期刊引用权重分级方案（文科 · 多学科 · 可微调）

> 版本 v1 · 2026-07 · 面向「整库锁定单学科」的中文文科知识库
> 本文件是给**另一个对话/AI 直接实现**用的完整规格。所有分级规则以「配置即代码」给出——**公开、纯数据、可被任意 AI 工具微调**（CC0）。
> 目标：让知识库不只是"显示期刊好不好看"，而是把期刊等级转成**可解释、可调的引用权重**，用于检索排序 / 证据强度 / 过滤 / 加权。

---

## 一、设计原则（务必遵守）

1. **目录优先，影响因子次之**。文科（尤其人文）影响因子普遍低且跨学科不可比，**主档位由"是否进入权威目录"决定**，影响因子只做同学科内微调。缺影响因子数据时方案照常成立。
2. **三层解耦**：客观目录数据 / 学科档案规则 / 权重解析，互不耦合。换数据不动规则，调权重不动数据，加学科只加一个档案。
3. **每学科一套系数**。法学与经管的"中外基准系数"正好相反（见下），一刀切必错一头。
4. **配置即代码**。分级体系是公开、注释齐全的 JSON/YAML，用户可用任意 AI 工具微调（改系数、加学科、换目录）。
5. **权重是"来源先验"，不是单篇质量**。journalWeight 只作为多因子之一参与排序，且每个权重可"点开看为什么"。
6. **整库锁定单学科**：一个知识库选定一个学科，全部条目用该学科档案评级（用户在库设置里选）。不做逐条自动判学科。

## 二、已核实的关键事实（2026，含变动预警）

> 均经联网对抗式核验（一手来源见文末）。这些事实直接决定了下面的设计取舍。

- **纯人文（A&HCI）"有影响因子数值、但没有基于影响因子的 Q1–Q4 分区"**。Clarivate 2024 年明确不为 25 个人文艺术学科做 JIF 排名（引用太少太慢，会产生大量并列、扭曲分位甚至空 Q 区）。→ **文史哲不能用 Q 分区排座次，只用"是否被 A&HCI/CSSCI 收录"分档。**
- **中科院期刊分区表已于 2026-03-27 官方停更**，2025 版是最后一版，第三方"新锐分区"权威性存疑。→ **不把中科院分区当主轴。**
- **Scopus SJR 在 scimagojr.com 免费、可导出 Excel**，单设"Arts & Humanities"领域，对人文社科覆盖比 Web of Science 更广。→ **SJR 作为唯一的外文分区数据源，免费内置。放弃 JCR（订阅、复杂）。**
- **CSSCI（南大核心）分"来源 / 扩展"两层**，2023-24 版来源 660 / 扩展 249；2025-26 版≈来源 674 / 扩展 261。文科最高本土标尺。
- **法学 CLSCI**（中国法学会 / 法学创新网发布）：2010 年 15 刊起，2024 年扩至约 26 刊，法学本土硬标尺，SSCI 在法学几乎不适用。
- **知网复合/综合影响因子**：复合口径含期刊+学位论文+会议，数值偏高；综合仅期刊。均出自知网《中国学术期刊影响因子年报》，非免费开放数据。→ **v1 不内置，留作 v2 经浏览器接入（见第九节）。**

## 三、总体架构

```
┌────────────────┐   ┌─────────────────────┐   ┌──────────────┐
│ 目录注册表      │ → │ 学科档案（10个）     │ → │ 权重解析器    │ → journalWeight ∈ [0,1]
│ catalogs       │   │ archetypes+disciplines│   │ resolver     │   (+解释链)
│ 客观期刊名单    │   │ 选目录/优先级/系数    │   │ priority-max │
└────────────────┘   └─────────────────────┘   └──────────────┘
        ↑ 离线数据文件（第八节）        ↑ 可微调配置（第七节）
```

- **目录注册表**：只存"哪本期刊在哪个目录的哪一级"，纯客观。
- **学科档案**：把目录信号翻译成本学科的档位与权重。10 个学科归入 4 个原型（archetype），复用档位映射，只覆盖系数与专属目录。
- **解析器**：期刊 → 命中目录 → 取优先级最高者定主档位 → 查分 → 影响因子微调 → 乘中外系数 → 归一。

**判定口径与边界（研究附注，实现/微调时参考）**
- 《中国社会科学》为全文科共同顶刊，纳入每个学科的 T1。
- **法学**：仅 CLSCI「三大权威」入 T1；T2 第一梯队 A 类（可另建 T2 表）：中外法学、法学家、清华法学、法商研究、法制与社会发展。
- **马理论**：求是为党刊理论旗舰；教学与研究、思想理论教育导刊在 CSSCI 为 B 类，但众多高校马院目录上抬为权威（**边界 T1**）。严格全国口径可只留前 4（中国社会科学、马克思主义研究、马克思主义与现实、求是）。
- **教育学**：清华大学教育研究、教育发展研究为 CSSCI-A 但 AMI 仅「核心」，未入 T1（宜作 T2）。
- **新闻传播**：即「四大刊」+ 中国社会科学；新闻记者/新闻界/当代传播等为 B 类，勿混入 T1。
- **中国语言文学**：当代语言学、方言为 T2；文艺研究名义属艺术学，因文艺理论最高档而纳入。
- **历史学**：考古学是**独立一级学科**（考古、考古学报、文物），若知识库把考古并入历史，另设考古 T1 子表；文史仅「核心」，未入 T1。
- **外国语言文学**：中国翻译为 CSSCI-B（翻译学方向旗舰），若单设翻译学可纳入其 T1；外国文学研究（AHCI 收录）按 A 类增补。
- **ISSN**：多数已核；现代外语(1003-6105)、外国文学研究(1003-7519) 为通行记载值，入库前建议再核刊号页。实际匹配以离线数据文件的权威 ISSN 为准。

## 六、权重解析算法（resolver）

对每个文献条目（已知知识库锁定学科 `activeDiscipline`）：

1. **期刊识别**：用 ISSN/eISSN 优先，回退到"归一化刊名"匹配目录数据（见 §7 matching）。
2. **取信号**：该期刊命中的所有 `catalog.level`。
3. **定主档位**：按 `resolution.priorityOrder` 从高到低，命中第一条 → 用该学科原型的 `map` 反查所属档位 T1–T6。（priority-max，避免多目录重复计分。）
4. **查基础分**：主档位 → `tiers[Tx]`。
5. **IF 微调**：`score = base * (1 + ifStrength * percentileInDiscipline(if))`；无 IF 数据或 ifStrength=0 时跳过。
6. **乘中外系数**：中文刊 × `coeff.cn`，外文刊 × `coeff.intl`。
7. **输出**：`journalWeight = clamp(score, 0, 1)`，附**解释链**（命中目录/级别/版本/是否用了IF）。

**跨学科可比性**：每学科都归一到 [0,1]="本领域相对地位"，故 law-T2(0.85) 与 business-T2(0.85) 语义一致（各自领域核心），比裸比影响因子公平。

**worked example**
- 《中国法学》→ law → clsci.权威 → T1 → 1.0 × cn1.0 = **1.00**
- 某 SSCI-Q1 管理刊 → business → T1 → 0.85(实为T1=1.0? 见map: SSCI.Q1在B原型=T1) → 1.0 × intl1.0 ×(1+0.2·分位) → 封顶 **1.00**
- 某 CSSCI 来源文学刊 → chinese_lit → T2 → 0.85 × cn1.0 ×(IF关闭) = **0.85**
- 命中预警名单 → 任意学科 → T6 → **0.05（标红）**

## 十、验收要点

- [ ] 期刊识别：ISSN 命中优先，刊名归一回退，未命中标"待确认"而非静默给普通档。
- [ ] priority-max：多目录命中时只按最高优先级定档，不重复加分。
- [ ] 纯人文档案**不出现** SSCI/JIF 分区档位。
- [ ] 中/外系数按学科生效（法学外0.70、经管外1.00 可验证）。
- [ ] 缺影响因子/缺分区数据时不报错、主档位照常。
- [ ] 每个 journalWeight 可展开解释链。
- [ ] 配置文件独立、注释齐全、可被 AI 工具直接改。
